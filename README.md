# ecommerce-sql-analysis
SQL-based E-commerce Sales Analysis project using MySQL with complex JOIN, subqueries, and query optimization.

# E-Commerce Sales Analysis using SQL

# Project Overview
This project analyzes an e-commerce database using SQL queries to extract business insights such as sales trends, customer spending, and product performance.

# Database Schema
The database consists of four main tables:

- customers
- products
- orders
- order_items

# Key SQL Concepts Used
- JOIN
- GROUP BY
- HAVING
- Subqueries
- Aggregations
- Indexing
- Query Optimization

# Business Analysis Queries
The project answers the following business questions:

1. What are the top selling products?
2. Which customers generate the highest revenue?
3. What is the monthly sales trend?
4. Which product categories generate the most revenue?

# Technologies Used
- MySQL
- phpMyAdmin
- SQL
